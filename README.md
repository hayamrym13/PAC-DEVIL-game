# 📖 About

PAC DEVIL is a modern arcade game inspired by the classic Pac-Man.

The player must collect all pellets while avoiding intelligent ghosts and dangerous devil traps including lasers, spikes, saws and void holes.

As the game progresses, levels become increasingly difficult with faster enemies and more hazards.

---

# ✨ Features

- 🎮 HTML5 Canvas Game
- 👻 AI Ghosts (BFS Pathfinding)
- 😈 Devil Traps
  - Lasers
  - Spikes
  - Rotating Saws
  - Void Holes
- ⚡ Power Pellets
- ❤️ Lives System
- 🏆 High Score
- 🎵 Sound Effects (Web Audio API)

- ⌨️ Keyboard Controls
- 📈 Progressive Difficulty
